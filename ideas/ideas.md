## Idea Lifecycle

```
1. Capture    → Quick note, rough idea
2. Develop    → Research, pros/cons, feasibility
3. Validate   → Prototype, POC, feedback
4. Implement  → Move to projects/ or tasks/
5. Archive    → Keep for reference
```

---

## How to Add Ideas

1. Copy template: `cp ../.templates/node-template.md ./idea-name.md`
2. Write initial thoughts in TL;DR
3. Develop as you research
4. Link to related projects/learnings
